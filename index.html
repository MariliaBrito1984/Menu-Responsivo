Inicio do projeto

Bem-vindo ao meu projeto! Decidi criar essa página como uma forma de praticar e demonstrar minhas habilidades em HTML, CSS e javaScript, ferramentas fundamentais para quem está começando no mundo da programação. Como sou nova na área, este projeto é uma oportunidade para aprender e explorar os conceitos básicos de desenvolvimento web. Meu objetivo aqui é criar uma página funcional, organizada e visualmente agradável.

Escolhi quatro temas principais para o site: Esporte, Pets, Saúde e Tecnologia. Cada um desses temas representa uma área de interesse, que pretendo desenvolver tanto em conteúdo quanto em estilo. O menu "Esporte", por exemplo, permitirá a criação de seções dinâmicas e interativas, onde posso experimentar diferentes estilos de layout para listas e artigos. Esportes também trazem oportunidades para exibir informações e imagens de maneira criativa, o que ajuda a desenvolver o design da página.
